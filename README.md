# DoubleList-Ruby

# Proyecto: Implementación de Estructura de Datos en Ruby

## 1. Descripción

Este proyecto consiste en la implementación de una **estructura de datos Lista Doblemente Enlazada (Double Linked List)** utilizando el lenguaje de programación **Ruby**.

La estructura permite almacenar elementos dinámicamente y realizar operaciones básicas como:

* Inserción de elementos
* Búsqueda de valores
* Eliminación de elementos
* Verificación de existencia de elementos
* Verificación de la existencia de la lista

El proyecto está desarrollado siguiendo el **paradigma de programación orientado a objetos (POO)**, haciendo uso de clases, objetos, atributos de instancia y métodos propios del lenguaje Ruby.

## 2. Requerimientos de Software

Para ejecutar el proyecto es necesario tener instalado:

### Lenguaje de programación

* **Ruby 3.x o superior**

Verificar instalación con:

```bash
ruby -v
```

### Sistema operativo compatible

* Windows
* Linux
* macOS

### Herramientas recomendadas

* Git
* Git Bash o Terminal
* Visual Studio Code (opcional)

No se requieren librerías externas ni configuraciones adicionales.
